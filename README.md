# Laravel Dump Server

Inspired by [beyondcode/laravel-dump-server](https://github.com/beyondcode/laravel-dump-server)

## Installation

```bash
composer require pkboom/laravel-dump-server --dev
```

## Usage

```bash
php artisan dump-server
```
